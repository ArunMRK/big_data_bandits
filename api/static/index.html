API: Big Data Bandits API for Deloton

Functionality Path
Get a ride with a specific ID /ride/{int:ride_id}
Get rider information (e.g. name, gender, age, avg. heart rate, number of rides) /rider/{string:user_id}
Get all rides for a rider with a specific ID /rider/{string:user_id}/rides
Delete a ride with a specific ID /ride/{int:ride_id}/delete
Get all rides for today /daily
Get all rides for a specific day /daily?date=dd-mm-yyyy
